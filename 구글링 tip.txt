구글링 잘하는법 Tip - 필요해서 정리

1.내가 무엇을 모르는지 정리부터 크게 3가지

-나는 지금 문법을 모른다 (구글에 자바스크립트 for 반복문)

-나는 지금 라이브러리 사용법을 모른다(구글에 리액트 usestate)

-나는 그냥 만드는 법을 모른다.(구글에 tap만드는법)

2.개발시 영어로 검색

한글로 검색시 자기공부하려고 필기해놓은글 . 강의노트 배껴 놓은거 등등 검색이 많이됨.

3.구글검색 문법 after

react pwa after:2020 = 2020년 이후

react pwa before:2020 = 2020년 이전

4.video 검색

리액트 설치 방법 ⇒ 유튜브 영상으로 보는게 더 괜찮음.

5.라이브러리 에러는

최신버전 라이브러리나 호환성 문제 등 github repo 에서 찿는거 추천

6.reddit 검색도 유용하다.

-최신소식 업데이트

-리액트 개발자 본인 무물

-좋은 해외강의

-좋은 프로젝트 샘플 등등 검색하기 유용

7.네이버 카페 . 하드코딩카페 .. 아두이노스토리 등등

프로젝트 도움이나. 고수분들에게 도움받을수 있음
